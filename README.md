## ✨ Что это?

**ÆON · FLOW** — одностраничный сайт, который превращает вашу дату рождения в захватывающую картину прожитого времени.  
Месяцы, недели, дни, часы, минуты, секунды, миллисекунды — и даже удары сердца и моргания.

Всё в премиальном тёмном дизайне с золотыми акцентами. Работает без сервера. Открывается в любом браузере.
